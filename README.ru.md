# ESPHome - Интеграция устройств ESPHome

![ESPHome Icon](static/ESPHome.png)

Нативная интеграция ESPHome API для osysHome с discovery, живой синхронизацией состояний, обратным управлением и связями с объектами.

## Документация

- [Руководство пользователя](docs/USER_GUIDE.ru.md)
- [Техническая документация](docs/TECHNICAL_REFERENCE.ru.md)
- [Индекс документации](docs/index.ru.md)

## Основные возможности

- Нативный ESPHome API client с логикой переподключения
- Ручное mDNS-обнаружение через `_esphomelib._tcp.local.`
- Хранение устройств и сущностей в базе данных
- Двусторонние связи между сущностями ESPHome и объектами osysHome
- Поддержка привязки как к свойствам, так и к методам
- Мост для подписок и сервисов Home Assistant
- Обновления административного интерфейса в реальном времени через WebSocket
